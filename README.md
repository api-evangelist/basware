# Basware

Basware is a cloud-based e-invoicing and purchase-to-pay platform that enables global enterprises to automate invoice and procurement processes. The platform provides REST APIs for submitting and receiving electronic invoices, managing purchase orders, processing accounts payable, and accessing network transaction data. Basware Network connects buyers and suppliers worldwide through a single integration point, supporting 70+ document formats with automatic conversion.

**Developer Portal:** https://developer.basware.com

**Website:** https://www.basware.com

## APIs

- **P2P API** — Accounts payable, procurement, master data, and user management
- **Network API** — E-invoice sending and receiving across the global Basware Network
- **Vault API** — Document archiving and compliance storage
- **Data Access API** — Reporting and data extraction
- **SmartPDF API** — PDF invoice capture and processing
